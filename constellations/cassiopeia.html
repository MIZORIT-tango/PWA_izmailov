Созвездие Кассиопеи

Кассиопея — характерное созвездие северного полушария, названное в честь царицы из греческой мифологии. Легко узнаётся по W-образной форме.

Основные характеристики:

  • Видимо в широтах от +90° до −20°
  • Лучшее время для наблюдения — осень
  • Площадь — 598 квадратных градусов
  • Ярчайшая звезда — Шедар (α Cas)

Интересные особенности:

Кассиопея содержит несколько примечательных объектов:

  • Остаток сверхновой звезды (Кассиопея A)
  • Рассеянные звёздные скопления
  • Используется для нахождения Полярной звезды
